# Installation
```
cd ~/ && rm -rf SuperDump
git clone --depth=1 https://github.com/solohackerzorg/Open-Source
cd ~/Open-Source
pip install requests
pip install futures
pip install rich
python sd.py
```


# Run

```
cd ~/Open-Source
python sd.py
```
