# Installation
```
cd ~/ && rm -rf SuperDump
git clone --depth=1 https://github.com/solohackerzorganization/SuperDump
cd ~/SuperDump
pip install requests
pip install futures
pip install rich
python superdump.py
```


# Run

```
cd ~/SuperDump
python superdump.py
```
